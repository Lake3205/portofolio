# Project Images

Place your project header images in this folder.

## Usage

To use an image in your projects, reference it like this in `src/data/projects.js`:

```javascript
headerImage: "/images/projects/your-image-name.jpg"
```

## Recommended Image Specifications

- **Dimensions**: 1200px Ã— 600px (2:1 aspect ratio)
- **Format**: JPG or PNG
- **File size**: Keep under 500KB for optimal loading

## Example

If you have an image named `my-project.jpg` in this folder, reference it as:

```javascript
headerImage: "/images/projects/my-project.jpg"
```
